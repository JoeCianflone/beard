Beard is an extremely pragmatic and utility-focused CSS framework. Actually, don't call it that. Beard focuses on developer speed, maintenance, and ease-of-use, versus "pretty" markup.

Features

Settings

Want to change or disable something in Beard? We make it easy to do so.

Reset & Base Styles

Beard uses a standard reset and has a small amount of sensible base styles to remove the most annoying defaults.

Objects

With roots in OOCSS, Beard comes with a small assortment of basic CSS objects like the infamous media object.

Components

You can now opt to pull in a bunch of handy pre-styled widgets and components.

Helpers

Beard ships with a ton of handy classes to help kickstart development and reduce the redundancy found in large CSS projects.

The Auto Grid

This AutoGrid is the easiest way to get started. Simply use a div with a .frame class and place any number of .blk elements as direct descendants and the AutoGrid will automatically figure out the column width. Each column is given a generous 30px of padding between to create a 60px wide gutter between columns.

12-column Grid

The 12-column grid is the gold standard in web layout. Beard ships with this standard and gives you responsive helper classes to change the layout when the viewport reaches a certain breakpoint. It also supports nested grids for increased flexibility.

Hint: To get the grid to maintain its ratios on the small breakpoint, use the .frame--stay modifier on your .frame wrapper.
