Installation

Beard is installed via Bower.

$ bower install beard

Adding Beard into your project

The Recommended Way

The best way to add Beard to your project is to copy the contents of beard/beard.scss to your main Sass file. From there, we recommend adding your site styles between the "Objects" and "Helpers" sections, like so:


    // Objects
    ...

    // Your styles go here

    // Helpers
    ...

Doing things this way allows for the best source ordering.

Note: You will have to change the import paths to the actual location as when they're copied into your main Sass file, they will have changed.

Dead Simple Installation

Or you could just add this shit to your Sass file:

@import '../bower_components/beard/beard'

We wouldn't do that, but if you need something quick, go ahead!
